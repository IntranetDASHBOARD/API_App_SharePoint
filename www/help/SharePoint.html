SharePoint Connector

Component Description

The SharePoint Connector is an application designed to enable SharePoint content to be surfaced within your iD intranet. It does this by allowing authorized users to import List Views from an externally installed SharePoint site and display it within an iD CMS content page. When this SharePoint content is updated, the List Views are simultaneously updated and reflected in SharePoint and iD allowing for the seamless display of SharePoint content via your intranet.

iD’s SharePoint Connector eliminates SharePoint silos around your organization and allows users to share valuable knowledge and information.

It is the first iD application developed using the iD API.

Functionality

  • Enables SharePoint content to be surfaced within your iD intranet
  • Acts as a window into SharePoint for intranet users, pulling in content and displaying it via iD’s easy to use interface
  • Display Task Lists and Document Libraries seamlessly within iD
